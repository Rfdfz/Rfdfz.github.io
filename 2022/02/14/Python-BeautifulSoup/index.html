Python-BeautifulSoup

发表于 2022-02-14 分类于 Python爬虫
本文字数： 4.3k 阅读时长 ≈ 4 分钟

灵活又方便的网页解析库，处理高效，支持多种解析器。

利用它不用编写正则表达式即可方便地实现网页信息的提取。

BeautifulSoup

解析库

解析器 使用方法 优势 劣势
Python标准库 BeautifulSoup(markup, “html.parser”) Python的内置标准库、执行速度适中、文档容错能力强 Python2.7.3 or 3.2.2前的版本中文容错能力差
lxml HTML 解析器 BeautifulSoup(markup, 'lxml') 速度快、文档容错能力强 需要安装C语言库
lxml XML 解析器 BeautifulSoup(markup, 'xml') 速度快、唯一支持XML的解析器 需要安装C语言库
html5lib BeautifulSoup(markup, 'html5lib') 最好的容错性、以浏览器的方式解析文档、生成HTML5格式的文档 速度慢、不依赖外部扩展

标准选择器

find_all()

语法：find_all(name, attrs, recursive, text, **kwargs)

作用：可根据标签名、属性、内容查找文档

find()

语法：find(name, attrs, recursive, text, **kwargs)

作用：find返回单个元素，find_all返回所有元素

find_parent()和find_parents()

语法：同上

作用：返回所有父节点或祖先节点

find_next_siblings()和find_next_sibling()

语法：同上

作用：find_next_siblings()返回前面所有兄弟节点，find_next_sibling()返回前一个兄弟节点

find_previous_siblings()和find_previous_sibling()

语法：同上

作用：find_previous_siblings()返回后面所有兄弟节点，find_previous_sibling()返回后一个兄弟节点

find_all_next()和find_next()

语法：同上

作用：find_all_next()返回前面所有符合条件的节点，find_next()返回前面第一个符合条件的节点

find_all_previous()和find_previous()

语法：同上

作用：find_all_previous()返回后面所有符合条件的节点，find_previous()返回后面第一个符合条件的节点

CSS选择器

通过select()直接传入CSS选择器即可完成选择

总结

  • 推荐使用lxml解析库，必要时使用html.parser
  • 标签选择筛选功能弱但速度快
  • 建议使用find()、find_all()查询匹配单个结果或者多个结果
  • 如果对CSS选择器熟悉建议使用select()
